    - Test: Transcribe whisper-cli failure — ffmpeg succeeds but whisper-cli fails, returns error containing "whisper-cli"
    - Test: Transcribe with Language set — whisper-cli args include "-l" and the language value
    - Test: Transcribe with empty Language — whisper-cli args do NOT include "-l"
    - Test: Temp directory is cleaned up after Transcribe returns (success and error paths)
  </behavior>
  <action>
Create `internal/transcribe/local.go`:

1. Define `localWhisper` struct with fields:
   - `BinaryPath string` — path to whisper-cli binary
   - `ModelPath string` — path to ggml model file
   - `Language string` — optional language hint
   - `execCmd func(ctx context.Context, name string, args ...string) *exec.Cmd` — injectable for testing

2. `newLocalWhisper(cfg config.TranscribeConfig) (*localWhisper, error)`:
   - Return error if `cfg.ModelPath == ""`: `"local provider requires model_path (set KAPSO_TRANSCRIBE_MODEL_PATH)"`
   - Set `execCmd` to default `exec.CommandContext` wrapper
   - Return populated struct

3. `(p *localWhisper) Transcribe(ctx context.Context, audio []byte, mimeType string) (string, error)`:
   - `os.MkdirTemp("", "kapso-whisper-*")` + `defer os.RemoveAll(dir)`
   - Write audio bytes to `dir/audio.ogg` with `os.WriteFile`
   - Run ffmpeg via `p.execCmd(ctx, "ffmpeg", "-y", "-loglevel", "error", "-i", rawPath, "-acodec", "pcm_s16le", "-ac", "1", "-ar", "16000", wavPath)` with `CombinedOutput()`
   - Build whisper-cli args: `-m`, ModelPath, `-f`, wavPath, `-otxt`, `-of`, outputPrefix. Append `-l`, Language if non-empty.
   - Run whisper-cli via `p.execCmd(ctx, p.BinaryPath, args...)` with `CombinedOutput()`
   - Read `outputPrefix + ".txt"` with `os.ReadFile`
   - Return `strings.TrimSpace(string(raw)), nil`
   - All errors wrapped with `fmt.Errorf` per project convention

Create `internal/transcribe/local_test.go` (package transcribe):

Use injectable `execCmd` to mock both ffmpeg and whisper-cli. The mock dispatches on the binary name:
- For ffmpeg mock: write a dummy WAV file to the output path found in args
- For whisper-cli mock: write transcript text to the `-of` prefix + ".txt" path found in args

Table-driven tests covering all behaviors listed above. Use `os.MkdirTemp` in tests to verify cleanup.

IMPORTANT anti-patterns to avoid:
- Do NOT use `-nt` flag (quality degradation bug whisper.cpp #2186)
- Do NOT use `exec.Command` — always `exec.CommandContext` for context cancellation
- Do NOT use individual `os.CreateTemp` — use `os.MkdirTemp` + `RemoveAll`
- Do NOT use `cmd.Stdout = &buf` — use `CombinedOutput()`
