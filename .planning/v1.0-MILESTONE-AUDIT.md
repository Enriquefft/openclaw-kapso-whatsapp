### Integration Issues (non-blocking)

**1. CacheTTL=0 disable path unreachable (low severity)**

`config.Validate()` at `config.go:345-347` unconditionally resets `CacheTTL <= 0` to `3600`. Since `main.go` calls `Validate()` before `transcribe.New()`, an operator setting `cache_ttl = 0` can never disable the cache. The factory's `CacheTTL == 0` skip-cache branch and `TestNewCacheTTLZeroSkipsCache` test are dead code in production.

Fix: Change zero-guard to `if c.Transcribe.CacheTTL < 0` (allow explicit 0) or add a `cache_enabled` boolean field.

**2. INFR-02 local provider timeout (low severity, documented decision)**

Cloud providers are fully protected by `retryTranscriber.timeout`. Local whisper.cpp receives `context.Background()` from `extract.go:53` and is not wrapped by retry. If ffmpeg/whisper-cli hangs, the pipeline goroutine blocks indefinitely. Documented design decision in Phase 3.
